TASK: Build the submission recording flow and the modules frontend — POST /v1/submissions persists upload metadata to Postgres after R2 upload, and users can browse modules via list and detail pages.

CONSTRAINTS:

- Backend: Hono on Cloudflare Workers, existing auth middleware, Supabase service client helper
- Frontend: React 19, TanStack Router, TanStack Query, shadcn/ui, existing upload mutation hook
- Submission is created AFTER the client successfully PUTs the file to R2 via the presigned URL
- The submission endpoint receives: moduleId (optional), objectKey, contentType, fileSizeBytes
- Frontend module pages use the existing design system (index.css tokens, shadcn/ui Card, Badge, Button)
- No new npm packages required
- Dashboard shell is a reusable layout for authenticated pages (modules, upload, future dashboard)
- File-based routing: TanStack Router auto-generates route tree from frontend/src/routes/

STEPS:

1. Backend — Create submission schema:

Create backend/src/schema/submission.ts
Import { z } from 'zod'
Export createSubmissionSchema:
  - moduleId: z.string().uuid().optional()
  - objectKey: z.string().min(1)
  - contentType: z.enum(['image/jpeg', 'image/png', 'image/webp'])
  - fileSizeBytes: z.number().int().min(1).max(20_971_520)
Export type CreateSubmission = z.infer<typeof createSubmissionSchema>

2. Backend — Create submissions route:

Create backend/src/routes/submissions.ts
Import Hono, Env, createServiceClient, authMiddleware, createSubmissionSchema
Create submissionsRouter = new Hono<{ Bindings: Env; Variables: { user: AuthVars } }>()
Add POST / handler:
  - Parse and validate body with createSubmissionSchema.safeParse
  - If invalid, return c.json({ error: result.error.issues }, 400)
  - const userId = c.get('user').id
  - const { moduleId, objectKey } = result.data
  - Build original_photo_url: `${c.env.R2_PUBLIC_BASE_URL}/${objectKey}` (add R2_PUBLIC_BASE_URL to Env type if missing)
  - const supabase = createServiceClient(c.env)
  - Insert into submissions: { user_id: userId, module_id: moduleId || null, original_photo_url, status: 'UPLOADED', review_type: 'AI' }
  - Return c.json({ submission: data }, 201)
Add GET /me handler:
  - const userId = c.get('user').id
  - const supabase = createServiceClient(c.env)
  - Select from submissions where user_id = userId, order by created_at desc
  - Return c.json({ submissions: data ?? [] }, 200)
Export submissionsRouter
